# Project Aura — Client

Minimal React frontend scaffold (Vite) for Project Aura.

Getting started

1. From the `client` folder install dependencies:

```powershell
cd "c:\Users\User\Documents\Project_Aura\client"
npm install
```

2. Start the dev server:

```powershell
npm run dev
```

Build for production:

```powershell
npm run build
npm run preview
```

Edit `src/App.jsx` to begin developing.
